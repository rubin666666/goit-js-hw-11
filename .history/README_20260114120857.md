# GOIT JS HW-11

Проект для домашнього завдання #11 курсу GOIT JavaScript.

## Установка залежностей

```bash
npm install
```

## Запуск в режимі розробки

```bash
npm run dev
```

## Збірка проекту

```bash
npm run build
```

## Предпросмотр

```bash
npm run preview
```

## Структура проекту

- `src/` - вихідні файли
  - `index.html` - основна HTML-сторінка
  - `main.js` - точка входу JavaScript
  - `css/` - файли стилів
  - `js/` - JavaScript модулі
    - `pixabay-api.js` - робота з API Pixabay
    - `render-functions.js` - функції рендеру
  - `img/` - зображення
  - `assets/` - інші ресурси
